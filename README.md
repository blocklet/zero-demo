## Develop locally

You can install Zero Server globally by:

`npm install -g zero`

You can install ABT Node globally by: 

`npm install -g @abtnode/cli`

### Run in development/debug mode

`abtnode dev`

### bundle as blocklet and test locally

```
npm run bundle
abtnode deploy _blocklet
```
